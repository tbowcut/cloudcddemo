DFS_Base (CHILD THEME OF ZURB_FOUNDATION)
----------------------------------------

There are two main purposes for this theme.

1) To be a utility (Base) theme that shares elements across all the Demo Framework scenarios.
2) Provide an out-of-the-box theme for administration (such as selecting a scenario UI)


ELEMENT SHARED
-------------------------------------------

- Panels layouts
- shared install config 
- Utility JavaScript functions
- Utility CSS functions (default Drupal overrides, for example)
- Generic shared page templates
- etc.
